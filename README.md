# Image Search Engine

A simple web-based image search engine that allows users to search for high-quality images using the Unsplash API.

## Features

- Search for images by keywords
- Responsive grid layout for image display
- Pagination with "Show more" functionality
- Direct links to original images on Unsplash
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Unsplash API

## Setup

1. Clone the repository:
   ```
   git clone https://github.com/your-username/image-search-engine.git
   ```

2. Navigate to the project directory:
   ```
   cd image-search-engine
   ```

3. Open `index.html` in your web browser.

## API Key

The project uses the Unsplash API. The API key is already included in `script.js`. For production use, consider obtaining your own API key from [Unsplash Developers](https://unsplash.com/developers).

## Usage

1. Enter a search term in the input field.
2. Click the "Search" button.
3. Browse the displayed images.
4. Click "Show more" to load additional results.
5. Click on any image to view it on Unsplash.
